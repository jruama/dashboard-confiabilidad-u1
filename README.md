# Dashboard de Confiabilidad Operacional — U1

Aplicación desarrollada en Python con Streamlit para analizar variables operativas y térmicas de una unidad de generación hidroeléctrica.

## Objetivo
Evaluar el comportamiento térmico y operativo de la unidad mediante análisis comparativo ANTES vs DESPUÉS.

## Tecnologías
- Python
- Streamlit
- Pandas
- Plotly

## Ejecución
```bash
streamlit run app.py
